# days-afterX_bot

**days-afterX_bot** — это Telegram-бот, разработанный для отслеживания количества дней, прошедших с момента определённых событий. Пользователи могут добавлять, удалять и просматривать события, а бот будет ежедневно присылать сводку.

## Основные функции (MVP)

- **Добавление событий**: Пользователи могут добавлять важные для них даты с помощью команды `/add`.
- **Просмотр событий**: Команда `/list` показывает все отслеживаемые события и количество дней с момента их последнего выполнения.
- **Удаление событий**: Команда `/delete` позволяет удалить событие из списка.
- **Сброс счетчика**: Интерактивная кнопка "Сделал!" под каждым событием в списке позволяет обнулить счетчик дней.
- **Ежедневные уведомления**: Бот автоматически рассылает ежедневную сводку по всем событиям в 8:00.

## Технологический стек

- **Язык**: Python 3.11+
- **Фреймворк для бота**: aiogram 3.x
- **База данных**: PostgreSQL
- **ORM**: SQLAlchemy
- **Управление зависимостями**: Poetry / pip
- **Контейнеризация**: Docker
