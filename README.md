# days-afterX_bot

Телеграм-бот для отслеживания количества дней, прошедших с определенной даты.

## Описание

Этот бот помогает пользователям запоминать и отслеживать важные события, считая дни, прошедшие с момента каждого из них.

## Основные команды

- `/start` - Регистрация и начало работы с ботом.
- `/add [название события] [дата]` - Добавить новое событие. Дату можно не указывать, тогда будет предложено выбрать из вариантов.
- `/list` - (В разработке) Показать все отслеживаемые события.
- `/delete` - (В разработке) Удалить событие.
